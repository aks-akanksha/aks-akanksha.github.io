# Gallery Images

Add your gallery images here. The gallery section will automatically display images from this folder.

## Supported Formats
- .jpg / .jpeg
- .png
- .gif
- .webp

## Naming Convention
You can name your images anything you like. The gallery will display them in alphabetical order.

## Recommended Sizes
- For best results, use square images (1:1 aspect ratio)
- Minimum recommended size: 400x400px
- Maximum recommended size: 2000x2000px

## How to Add Images
1. Simply add your image files to this folder
2. Commit and push to GitHub
3. The gallery will automatically update to show your images
